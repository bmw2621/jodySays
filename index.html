Jody Says API

Usage

GET Request:

Make request to http://jodysays.gq/say/ followed by a url encoded string.

Response will be a jpeg of Jody saying your string


Post Request:

Make request to http://jodysays.gq/ with "text" parameter as a string

Response be a JSON object with text atrribute giving html link to the above GET request. This was developed to support Slackbot.
